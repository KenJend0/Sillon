Sillon.
Ce qui reste quand la musique s'arrête.

Le sillon est le creux gravé dans le vinyle — sans lui, pas de son. C'est aussi la trace qu'on laisse, le journal qu'on écrit. Le mark existant est déjà un sillon vu de face. Rien ne change, tout s'aligne.

nomexploration
Pourquoi ce nom fonctionne

Le mark était déjà un sillon.

La courbe continue existante représente une onde audio — c'est visuellement identique à un sillon de vinyle vu de profil. Aucun redesign nécessaire : le nom donne enfin un sens précis à l'image.

Le sens retrouvé. "Waveform" décrivait la forme — Sillon décrit l'objet. Le mark n'est plus une abstraction : c'est précisément ce qu'un sillon ressemble vu de côté, les crêtes et les creux gravés par l'aiguille.

Le double sens. Sillon = trace dans le vinyle, mais aussi "laisser un sillon" en français — laisser une empreinte durable. C'est exactement ce que fait l'app : transformer chaque écoute en mémoire durable.

Unicité. Quasi-absent en tech. Aucune app musicale, aucun service de streaming, aucune startup n'utilise ce nom. Les domaines .app et .fm sont libres.

Périmètre du changement

Ce qui change, ce qui reste.

Le rebranding est chirurgical. Le système visuel est solide — on touche uniquement ce qui porte le nom.
